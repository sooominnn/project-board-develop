# soomin-project-board-develop
자바 + 스프링부트 관련 기술로 게시판 만들기 - develop 버전

# 개발환경
1. IDE: IntelliJ IDEA Community
2. Spring Boot 3.1.1
3. JDK 17
4. mysql
5. Spring Data JPA
6. Thymeleaf

# 게시판 주요기능 
1. 회원가입
2. 로그인/로그아웃
3. 회원조회
4. 회원 정보 수정
5. 게시글 생성, 조회, 수정, 삭제
6. 게시글 목록 조회 페이징처리
7. 댓글 생성, 조회, 수정, 삭제
8. 해시태그 검색
